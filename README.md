# 🍽️ Recipe Explorer Dashboard

Ein interaktives Dashboard zur Rezeptsuche und -analyse – gebaut mit Vanilla JavaScript und der DummyJSON Recipes API.

---

## 📁 Projektstruktur

```
/
├── index.html
├── styles.css
└── scripts.js
```

<br>

## ✨ Features

- Alle Rezepte werden beim Start automatisch von der API geladen
- Echtzeit-Suche nach Rezeptnamen (Button & Enter-Taste)
- Klickbare Tabellenzeilen öffnen ein Modal mit Details (Zutaten, Zubereitung, Bild, Zeit)
- Statistiken: Gesamtanzahl, Durchschnittliche Zubereitungszeit, Anzahl Küchen
- Balkendiagramm der Rezeptverteilung nach Küche
- Fehlerbehandlung und Ladeanzeige

<br>

## 🛠️ Technologien

- **JavaScript (ES6+)** – async/await, fetch(), IIFE, DOM-Manipulation, Event Listener, Array-Methoden
- **HTML5** – semantisches Markup, Modal-Struktur
- **CSS3** – Grid, eigenes Balkendiagramm, Modal-Styling
- **DummyJSON Recipes API** – `https://dummyjson.com/recipes`
